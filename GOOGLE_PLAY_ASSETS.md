# Google Play Store Assets Guide

## App Icon Requirements

### 1. **App Icon (Required)**
- **Size**: 512x512 pixels
- **Format**: PNG (32-bit with alpha)
- **Design**: Simple, recognizable, works at small sizes
- **Content**: Should represent typing/learning/autism support
- **Colors**: Bright, friendly colors (consider autism-friendly palette)
- **Avoid**: Text, photos, complex gradients

**Suggested Design Elements:**
- Keyboard keys
- Letter symbols (A, B, C)
- Colorful/rainbow theme (represents autism community)
- Friendly character/pet
- Simple geometric shapes

### 2. **Feature Graphic (Required for Play Store)**
- **Size**: 1024x500 pixels
- **Format**: PNG or JPEG
- **Purpose**: Banner shown on Play Store listing
- **Content**: App name, tagline, visuals

**Example Text:**
```
Autism Typing Tutor
AI-Powered Learning for Kids
```

### 3. **Screenshots (Required)**

**Phone Screenshots** (Minimum 2, Maximum 8):
- **Size**: 1080x1920 pixels (9:16 aspect ratio)
- **Capture**:
  1. Home screen
  2. Word practice with picture
  3. Comprehension question (2 pictures)
  4. Progress/achievements screen
  5. Parent dashboard

**Tablet Screenshots** (Optional but recommended):
- **Size**: 1920x1200 pixels (16:10 aspect ratio)
- Same content as phone, adapted for tablet

### 4. **Splash Screen**
- Already generated by Capacitor
- Located in: `android/app/src/main/res/drawable-*/splash.png`
- Can customize with your branding

## Google Play Console Checklist

### App Details
- [ ] App title
- [ ] Short description
- [ ] Full description
- [ ] App icon (512x512)
- [ ] Feature graphic (1024x500)
- [ ] Screenshots (min 2 phone)
- [ ] Category
- [ ] Contact email
- [ ] Privacy policy URL

### Content Rating
- [ ] Complete IARC questionnaire
- [ ] Declare "Made for Kids"
- [ ] No ads confirmation
- [ ] No in-app purchases

### Store Presence
- [ ] Main Store Listing (English US required)
- [ ] Optional: Translations for other languages

### Pricing & Distribution
- [ ] Free or Paid (Recommend: Free)
- [ ] Available countries (Start with US, expand later)
- [ ] Device compatibility (Phone + Tablet)

### App Releases
- [ ] Internal testing (optional)
- [ ] Closed testing (optional)
- [ ] Open testing (beta)
- [ ] Production release

## Icon Design Recommendations

**Color Palette (Autism-Friendly):**
- Primary: Bright blue (#4A90E2)
- Secondary: Soft green (#4CAF50)
- Accent: Warm orange (#FF9800)
- Background: Light purple (#E1BEE7)

**Design Tools:**
- Canva (easy, templates available)
- Figma (professional, free)
- Adobe Illustrator (advanced)
- GIMP (free Photoshop alternative)

**Icon Generator Tools:**
- https://www.appicon.co/
- https://icon.kitchen/
- https://romannurik.github.io/AndroidAssetStudio/

## Next Steps

1. **Create App Icon** (512x512 PNG)
2. **Create Feature Graphic** (1024x500 PNG)
3. **Take Screenshots** (use emulator or device)
4. **Sign APK** (generate keystore)
5. **Create Google Play Developer Account** ($25 one-time fee)
6. **Upload and submit for review**

Typical review time: 1-7 days
